Internet Explorer is harmful to the mental health of developers

Jokes aside, it is hard work to get even the simplest modern website to work in Internet Explorer without resorting to hacks, workarounds, pollyfills and outmoded development techniques because Microsoft have never kept pace with support for the open standards supported by other browser vendors and relied upon by developers.

If this page starts showing up in my analytics with any regularity I might put in the effort to make it work, but for now please consider using Chrome, Safari, Firefox, Edge or Opera.
